# Belly_Button_Biodiversity

This project tells the torrid tale of bacteria samples taken from belly buttons in combination with their washing frequency.

![](Belly_Button_Biodiversity/bacteria.jpg)

I utilized a combination of html, css, python, flask, javascript, d3/plotly to create the visualizations and glean the data from sql.
It's deployed on heroku. Check it out here: https://tots-an-app.herokuapp.com.
